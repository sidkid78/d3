# Dwellingly - Commission Infrastructure™

**Dwellingly** is a premium, high-performance "Commission Infrastructure" layer designed to protect real estate income in the post-NAR settlement era. We transition agents from an implicit trust model to an explicit, verifiable digital standard for income protection.

## ✨ "Liquid Glass" Aesthetic

The platform features a state-of-the-art **Liquid Glass** design system, engineered for a professional and secure atmosphere:

- **Glassmorphism**: High-fidelity translucent panels with multi-layered Gaussian blur backdrops.
- **Ambient Intelligence**: Dynamic, fixed radial glows that create depth and visual hierarchy without clutter.
- **Space Grotesk Typography**: Ultra-clean, high-precision font matching the "Infrastructure" branding.
- **Micro-Animations**: Subtle transitions and hover states that make the interface feel responsive and "alive".
- **Emerald Primary**: A curated emerald teal (`#308875`) representing institutional security and financial growth.

## 🚀 Key Features

- **30-Second Handshake**: Legally binding buyer representation agreements signed in under a minute, optimized for mobile-first agents.
- **Smart Certificates**: Immutable PDF commission certificates generated via Gemini AI and jsPDF.
- **Public Trust API**: Secure verification endpoints (`/verify/[id]`) allowing Title Companies and Listing Agents to validate agreements instantly.
- **Premium Command Center**: A high-performance dashboard with real-time analytics for Sent, Signed, and Pending agreements.
- **Economic Shield**: Full subscription Lifecycle management and automated billing via Stripe integration.

## 🛠 Technical Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router, Server Actions)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Vanilla CSS Architecture)
- **Backend/Auth**: [Supabase](https://supabase.com/) (PostgreSQL + RLS + real-time capabilities)
- **AI Engine**: [Google GenAI SDK](https://ai.google.dev/gemini-api/docs) (`gemini-3-flash-preview`)
- **Document Logic**: [jsPDF](https://github.com/parallax/jsPDF) for high-fidelity PDF generation.
- **Commerce**: [Stripe](https://stripe.com/) for secure platform monetization.

## 🏗 Development Status

The project is currently in the late **"Premium Polish"** phase.

- **Supabase Bypass**: A `SUPABASE_BYPASS` mode is integrated for rapid UI/UX prototyping.
- **Production Readiness**: Performance optimized, fully responsive, and compliant with modern web standards.

## 📖 Getting Started

1. **Install Architecture**:

   ```bash
   pnpm install
   ```

2. **Environment Configuration**:
   Copy `.env.example` to `.env.local` and configure your credentials.

3. **Launch Infrastructure**:

   ```bash
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) to enter the platform.
